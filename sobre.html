PROJETO - Plataforma Gamer

Sobre o site

Bem-vindo à nossa plataforma de jogos online gratuitos! Aqui, você pode jogar jogos divertidos diretamente no navegador, sem necessidade de downloads. Este projeto integra conhecimentos de Desenvolvimento de Sistemas Web e Organização de Sistemas de Computação, utilizando Assembly para o desenvolvimento dos jogos e JavaScript para a interação com o navegador.

Experimente jogos como Batalha Naval, Snake Game e Pac-Man todos acessíveis de forma prática e divertida, com uma interface moderna e responsiva.

Jogos Disponíveis

  • Batalha Naval
  • Snake Game
  • Pac-Man

Como Jogar

Jogar é simples! Acesse a plataforma e selecione o jogo. Use as teclas de direção, espaço ou clique do mouse para interagir com os jogos. Não há necessidade de instalar nada!

Controles Específicos

  • Batalha Naval: Use as teclas de direção para mover o cursor e a tecla de espaço para fazer seleções.
  • Snake Game: Controle a cobrinha com as teclas de seta e evite as paredes e seu próprio corpo.
  • Pac-Man: Navegue pelas células com as teclas de direção e marque as minas com o clique do mouse.

Tecnologias Utilizadas

  • HTML: Estrutura das páginas.
  • CSS: Estilo e design visual.
  • JavaScript: Controle de interação e exibição dos jogos.
  • Assembly: Desenvolvimento dos jogos em baixo nível.
